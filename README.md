# spring-boot-custom-autoconfig

Clone this repository:

```bash
git clone https://github.com/deepaksorthiya/spring-boot-custom-autoconfig.git
cd spring-boot-custom-autoconfig
```

Start Service:

```bash
mvn spring-boot:run
```
